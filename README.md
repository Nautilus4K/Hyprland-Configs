# Hyprland-Configs
 My Hyprland config, along with SDDM and Quickshell

> [!IMPORTANT]
> This is bundled with things like kitty config, zsh shell config, etc...
> Therefore, it is recommended to not fuck around with them, pretty please?

There are multiple hardcoded paths, you can find them in quickshell/themes.json, quickshell/shell.qml (Especially the config application part), and sddm/Main.qml.

You can change them if you want, but I advise against it.
However, the sample fastfetch config in quickshell/shell.qml do have a hardcoded path that I think is better to mention, and its in ~/Pictures/miku.jpg, it is quite hard to make out, but once you found it, it will be easier to change.
